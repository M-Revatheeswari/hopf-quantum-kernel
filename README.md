# Hopf Algebra q-Deformed Quantum Kernel

Code accompanying the paper:
"A Hopf Algebra q-Deformed Feature Map for Quantum Kernels: Construction,
Verified Properties, and an Honest Account of What Does and Does Not Hold"

## Requirements
numpy, matplotlib, scipy, scikit-learn

## Reproduce all results
python hopf_experiments.py

All experiments use a fixed random seed (42) and are fully deterministic.
This script reproduces every number and figure reported in the paper,
including the reported negative findings (Sections 6-8).
